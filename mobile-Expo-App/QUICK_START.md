# دليل البدء السريع - تطبيق جيم روم

## خطوات إنشاء APK بسرعة:

### الطريقة الأولى: استخدام الملف التلقائي
```bash
# شغل الملف التلقائي
BUILD_APK.bat
```

### الطريقة الثانية: خطوة بخطوة

1. **تثبيت المتطلبات:**
```bash
npm install -g @expo/cli eas-cli
```

2. **تثبيت التبعيات:**
```bash
npm install
```

3. **إنشاء الأيقونات:**
```bash
npm run create-icons
```
- افتح `icon-generator.html` في المتصفح
- حمل كل الأيقونات إلى مجلد `assets`

4. **تسجيل الدخول:**
```bash
eas login
```

5. **بناء APK:**
```bash
npm run build:android
```
