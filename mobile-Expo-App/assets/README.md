# مجلد الأيقونات والصور

ضع هنا الملفات التالية بعد إنشائها من icon-generator.html:

- `icon.png` (1024x1024) - أيقونة التطبيق الرئيسية
- `adaptive-icon.png` (1024x1024) - أيقونة Android التكيفية  
- `splash-icon.png` (512x512) - صورة شاشة البداية
- `favicon.png` (32x32) - أيقونة الويب

## كيفية إنشاء الأيقونات:

1. شغل الأمر: `npm run create-icons`
2. افتح ملف `icon-generator.html` في المتصفح
3. سيتم تحميل شعار الموقع تلقائياً
4. اضغط على أزرار التحميل لكل أيقونة
5. ضع الملفات المحملة في هذا المجلد

الأيقونات ستكون بخلفية بنفسجية (#7c3aed) مع شعار الشركة في المنتصف.
