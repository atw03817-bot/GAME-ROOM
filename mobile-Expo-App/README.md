# تطبيق جيم روم - Mobile App

تطبيق موبايل لمتجر جيم روم باستخدام Expo و React Native WebView.

## المميزات
- عرض الموقع الكامل داخل التطبيق
- شاشة ترحيب مخصصة مع شعار الشركة
- متصفح داخلي للمتاجر الخارجية
- تصميم متجاوب مع الألوان الرسمية للشركة

## إعداد التطبيق

### 1. تثبيت المتطلبات
```bash
npm install -g @expo/cli eas-cli
```

### 2. تثبيت التبعيات
```bash
cd mobile-Expo-App
npm install
```

### 3. إنشاء الأيقونات
```bash
npm run create-icons
```
ثم افتح `icon-generator.html` في المتصفح وحمل الأيقونات إلى مجلد `assets`

### 4. تشغيل التطبيق للتطوير
```bash
npm start
```

### 5. بناء APK للتوزيع
```bash
# تسجيل الدخول إلى Expo
eas login

# بناء APK
npm run build:android
```

## الملفات المهمة

- `App.js` - الملف الرئيسي للتطبيق
- `config.js` - إعدادات التطبيق (الألوان، الرابط، النصوص)
- `app.json` - إعدادات Expo
- `eas.json` - إعدادات البناء

## تخصيص التطبيق

### تغيير الموقع المستهدف
عدل `WEBSITE_URL` في ملف `config.js`

### تغيير الألوان
عدل `COLORS` في ملف `config.js`

### تغيير اسم التطبيق
عدل `APP_NAME` في ملف `config.js` و `name` في ملف `app.json`

## بناء APK

1. تأكد من تسجيل الدخول: `eas login`
2. شغل الأمر: `npm run build:android`
3. انتظر انتهاء البناء (قد يستغرق 10-15 دقيقة)
4. حمل APK من الرابط الذي سيظهر
