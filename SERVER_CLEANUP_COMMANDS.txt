أوامر تنظيف السيرفر - نسخ والصق بالترتيب:

# 1. الانتقال للمجلد والسحب
cd /var/www/mobile-store
git pull origin main

# 2. تنظيف قاعدة البيانات (حذف الطلبات وإصلاح العملاء)
cd backend
node CLEAN_DATABASE.js

# 3. إنشاء حساب المدير مع الاسم الصحيح
node CREATE_ADMIN_WITH_NAME.js

# 4. إعادة تشغيل السيرفر
cd ..
pm2 restart all

# 5. التحقق من النتيجة
pm2 logs --lines 5

بعد تطبيق هذه الأوامر:
- سيظهر الداشبورد 4 عملاء بدلاً من 1236
- سيتم حذف جميع الطلبات الموجودة
- حساب المدير سيظهر باسم "مدير"
